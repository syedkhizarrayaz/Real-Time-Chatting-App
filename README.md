# Real-Time Chat App
The real-time Chat app was developed using the MERN stack along with Socket IO.
 
# Run Client and Socket
- npm install
- npm start

# Run Api
- create an .env file containing MONGO_URL variable, which will contain your Mongo API key.
- npm install
- npm start
